# 🍣 鮨サクセス 〜目指せミシュラン三つ星〜

パワプロの「サクセスモード」をベースにした、お寿司屋さん育成 × お店運営アクションゲーム。
ブラウザだけで動きます（ビルド不要）。営業パートは **Three.js** による3D。
3D描画ライブラリ（Three.js）は CDN から読み込むため、プレイには **ネット接続** が必要です。

## 遊び方

`index.html` をブラウザで開くだけ。

> 一部ブラウザでは `file://` 直開きでも動作しますが、うまく動かない場合はローカルサーバー経由で開いてください。
> ```bash
> python3 -m http.server 8123
> # → http://localhost:8123 を開く
> ```

## ゲームの流れ

1. **ポジション選択** — 寿司職人 🍣 / キッチン担当 🍳 / フロアスタッフ 🍵 / マネージャー 📋 から1つ。
   得意ステータスが伸びやすく（能力アップ割引）、**ポジション専用イベント**や**営業の中身**が変わる。
2. **育成パート（サクセス）** — 全20週。コマンドはすべて**トレードオフ**：
   - ホール練習＝接客力・客評価↑（ストレス＋）／キッチン補助＝作業力・同僚評価↑（ストレス＋）
   - メニュー暗記＝経験点大・判断力↑（ストレス大）／クレーム対応訓練＝成功でメンタル・接客力↑（失敗リスク）
   - 休む＝体力回復／遊ぶ＝やる気・ストレス解消／同僚と話す＝同僚評価・コミュ力↑
   - 「**能力アップ**」で経験点を5能力（作業力・接客力・判断力・メンタル・コミュ力）に自由配分（パワプロ式）。
   **やる気**が経験点効率に直結し、**ストレス**は100で倒れる。**メンタル**が高いとストレスが溜まりにくく、
   **コミュ力**が高いと評価が上がりやすい。判定つき選択肢は**成功率が表示**される（判断力などで変動）。
3. **対立軸：大将評価 × 同僚評価** — イベントの選択で両者がせめぎ合う。
   大将評価が高いと引き立てイベント、低いと理不尽イベント。同僚評価が高いと支援イベント、低いと孤立イベント。
4. **営業パート（5・10・15・20週目）** — ポジションで内容が変化（職人/キッチン/フロア＝3Dアクション、
   マネージャー＝采配・配置型）。結果は**客評価・大将評価・同僚評価**にも反映される。
5. **エンディング分岐** — 能力・評価・ストレスの組み合わせで6種に分岐：
   🌟現場のエース／👔大将のお気に入り（孤立）／🤝現場の味方／🔥燃え尽き退職／🚪転職成功／🍵なんとか生存。
   店の格（ミシュラン）はサブ評価として表示。

## ファイル構成

```
index.html           画面レイアウト
css/style.css        スタイル（和風・寿司屋テイスト）
js/types/types.js    型定義（JSDoc typedef）
js/data.js           ステータス / ポジション / レシピ / アイテム / ミシュラン
js/data/balance.js   調整値（初期値・倍率・しきい値・能力アップコスト）
js/data/commands.js  育成コマンド定義
js/data/events.js    イベント定義（対立軸・関係性・ポジション専用）
js/data/endings.js   エンディング分岐定義
js/logic/effects.js  効果適用・成功判定（メンタル軽減・コミュ増幅・やる気倍率）
js/logic/events.js   イベント抽選（評価条件・重み付け）
js/logic/endings.js  エンディング判定
js/chara.js          キャラ立ち絵（主人公・大将・彼女・ライバル）とセリフ
js/cooking.js        営業（職人/キッチン/フロア・Three.js 3Dアクション）
js/manager.js        営業（マネージャー・采配/配置型）
js/game.js           メインコントローラ（画面遷移・コマンド実行・UI）
```

## ポジション特性

| ポジション | 得意（能力アップ割引） | 営業ボーナス |
|---|---|---|
| 寿司職人 🍣 | 作業力 | 寿司系メニューのスコア +15% |
| キッチン担当 🍳 | メンタル | 営業の制限時間 +8秒 |
| フロアスタッフ 🍵 | 接客力 | 客の我慢 +25% / チップ +30% |
| マネージャー 📋 | 判断力 | 営業後の店評価アップ +20% |
