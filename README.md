# 🍣 鮨サクセス 〜目指せミシュラン三つ星〜

パワプロの「サクセスモード」をベースにした、お寿司屋さん育成 × お店運営アクションゲーム。
ブラウザだけで動きます（ビルド不要）。営業パートは **Three.js** による3D。
3D描画ライブラリ（Three.js）は CDN から読み込むため、プレイには **ネット接続** が必要です。

## 遊び方

`index.html` をブラウザで開くだけ。

> 一部ブラウザでは `file://` 直開きでも動作しますが、うまく動かない場合はローカルサーバー経由で開いてください。
> ```bash
> python3 -m http.server 8123
> # → http://localhost:8123 を開く
> ```

## ゲームの流れ

1. **ポジション選択** — 寿司職人 🍣 / キッチン担当 🍳 / フロアスタッフ 🍵 / マネージャー 📋 から1つ。
   得意ステータスが伸びやすく（能力アップ割引）、**ポジション専用イベント**や**営業の中身**が変わる。
2. **育成パート（サクセス）** — 全20週。訓練（猛特訓 / 通常 / 軽い / 得意稽古）で **経験点** を稼ぎ、
   「**能力アップ**」画面で経験点を5能力に**自由に振り分けて**伸ばす（パワプロ式）。
   休養・食べ歩き・アイテム・ポジション別ランダムイベントもあり。
3. **営業パート（5・10・15・20週目）** — ポジションで内容が変化。育成した**全ステータスが影響**：
   - **寿司職人 / キッチン / フロア** … 3DキッチンでシェフをWASD・矢印（スマホは十字キー）で操作。
     お皿を取り→手順どおり組み立て→提供口へ。技術＝早業・創作＝ひらめき神握り・知識＝高得点メニュー
     ・接客＝我慢/チップ・スピード＝移動/制限時間。ポジションで出る品やペースが変わる。
   - **マネージャー** … **采配・配置型**。テーブルに来た客へスタッフを配置し、「急がせる/丁寧に/檄」の
     采配で店を回す。知識＝処理効率・接客＝満足/チップ・技術＝品質・創作＝名物テーブル。
4. **最終評価** — 店の評価と総合ステータスから、ミシュラン評価（無星 → ビブグルマン → ★ → ★★ → ★★★）を判定。

## ファイル構成

```
index.html        画面レイアウト
css/style.css     スタイル（和風・寿司屋テイスト）
js/data.js        ポジション / レシピ / アイテム / イベント / 訓練・能力アップ定義
js/chara.js       キャラ立ち絵（主人公・大将・彼女・ライバル）とセリフ
js/cooking.js     営業（職人/キッチン/フロア・Three.js 3Dアクション）
js/manager.js     営業（マネージャー・采配/配置型）
js/game.js        メインコントローラ（育成・経験点振り分け・画面遷移・評価集計）
```

## ポジション特性

| ポジション | 得意 | 営業ボーナス |
|---|---|---|
| 寿司職人 🍣 | 技術 | 寿司系メニューのスコア +15% |
| キッチン担当 🍳 | スピード | 営業の制限時間 +8秒 |
| フロアスタッフ 🍵 | 接客 | 客の我慢 +25% / チップ +30% |
| マネージャー 📋 | 知識 | 営業後の店評価アップ +20% |
