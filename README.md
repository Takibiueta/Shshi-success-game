# 🍣 鮨サクセス 〜目指せミシュラン三つ星〜

パワプロの「サクセスモード」をベースにした、お寿司屋さん育成 × お店運営アクションゲーム。
ブラウザだけで動きます（外部ライブラリ・ビルド不要）。

## 遊び方

`index.html` をブラウザで開くだけ。

> 一部ブラウザでは `file://` 直開きでも動作しますが、うまく動かない場合はローカルサーバー経由で開いてください。
> ```bash
> python3 -m http.server 8123
> # → http://localhost:8123 を開く
> ```

## ゲームの流れ

1. **ポジション選択** — 寿司職人 🍣 / キッチン担当 🍳 / フロアスタッフ 🍵 / マネージャー 📋 から1つ。
   それぞれ得意ステータスが伸びやすく、営業で固有ボーナスがある。
2. **育成パート（サクセス）** — 全12週。練習でステータス（技術・スピード・知識・接客・創作）を伸ばす。
   体力を消費し、休養や食べ歩き、ランダムイベントもあり。
3. **営業パート（パワプロの“試合”に相当 / Overcookedスタイル）** — 4・8・12週目に発生。
   上から見たキッチンでシェフ🧑‍🍳を自分で操作する。**WASD / 矢印キー**で移動、**スペース / Enter**（スマホは画面の十字キー＋ボタン）で作業。
   まず**お皿台🍽️**でお皿を取り（同時に作れるのは1皿だけ）、食材台・調理台（包丁・出汁・揚げ）に近づいて作業すると工程がお皿に積まれる。
   注文どおりに組み立てて**提供口🛎️**まで運ぶと提供完了（お皿はまた取り直し）。間違えたら**片付け🗑️**台でやり直し。
   時間差で来店する客の我慢ゲージ・コンボ・チップを管理する。育成したステータスが各パラメータ（移動速度・制限時間・我慢・チップ・スコア）に影響。
4. **最終評価** — 店の評価と総合ステータスから、ミシュラン評価（無星 → ビブグルマン → ★ → ★★ → ★★★）を判定。

## ファイル構成

```
index.html        画面レイアウト
css/style.css     スタイル（和風・寿司屋テイスト）
js/data.js        ポジション / レシピ / イベント / 評価などのデータ定義
js/cooking.js     営業パート（アクションミニゲーム）
js/game.js        メインコントローラ（画面遷移・育成・評価集計）
```

## ポジション特性

| ポジション | 得意 | 営業ボーナス |
|---|---|---|
| 寿司職人 🍣 | 技術 | 寿司系メニューのスコア +15% |
| キッチン担当 🍳 | スピード | 営業の制限時間 +8秒 |
| フロアスタッフ 🍵 | 接客 | 客の我慢 +25% / チップ +30% |
| マネージャー 📋 | 知識 | 営業後の店評価アップ +20% |
